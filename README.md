# APITable Widget - RepeatDelete (重复项合并删除工具)

本项目是一个为 [APITable (维格表)](https://apitable.com/) / AITable 开发的自定义小程序（Widget）。主要用于在表内快速筛选出重复条目，并将重复的多行数据智能合并为一行，最后自动删除多余的重复记录。

## 🌟 功能特性

- **灵活判断重复条件**：提供下拉菜单指定用来判断重复的字段（默认自动选择表格第一列）。
- **直观的重复项列表**：点击“查找重复项”后，自动筛选并按重复值将条目分组展示在列表中，一目了然。
- **快捷的组选择操作**：
  - 列表左侧提供多选框，每次勾选代表选中该重复值下的**所有关联记录**。
  - 提供 **全选**、**反选**、**取消选择** 按钮，方便大规模批量操作。
- **智能的合并策略**：
  点击“合并”按钮后，程序会执行复杂的合并逻辑，将一组重复行的内容智能整合至第一条数据中，随后删除其余行。
  *合并规则基于具体的表格字段类型自动判断：*
  - **单选、单行文字、多行文字、日期、数字等字段**：以最后编辑的有效数据为准（优先保留有值的记录）。
  - **多选、成员、链接引用字段、附件字段**：将该组内所有记录的内容全部整合，并自动去重，保留所有的唯一值（合并成一个聚合大列表）。
  - **其他常规字段**：默认以最后有效的数据为准。

## 🚀 开发与部署说明

本项目基于 `@apitable/widget-sdk` 开发，使用 React 构建。

### 1. 环境准备
确保你的本地环境已安装 [Node.js](https://nodejs.org/) 以及包管理工具 (`npm` 或 `yarn`)。

### 2. 安装依赖
克隆或下载本代码仓库后，在根目录执行以下命令安装依赖：

```bash
npm install
# 或者
yarn install
```

### 3. 本地启动开发环境
使用 APITable 官方的开发者工具启动小程序即可进入开发与调试模式：

```bash
npm start
# 或者
yarn start
```
*启动后，在你的 APITable 工作区中添加小程序，选择「本地开发模式」，即可实时在表中预览你的合并操作功能。*

### 4. 发布上线
开发测试完成后，执行 release 可将小程序打包并上传发布至你的空间站：

```bash
npm run release
# 或者
yarn release
```

## 📝 贡献与定制

该小程序核心合并逻辑位于 `src/utils.ts` 中，UI 交互逻辑位于 `src/index.tsx` 中。
你可以根据业务定制自己独有的合并与剔除逻辑（例如特定字段按金额求和合并等）。欢迎根据自己的需求进行二次开发！
